Comprendre le Web Moderne

Une synthèse claire des grands principes du web, ses risques et ses standards.

📜 Timeline du Web

  • 1991 : Lancement du World Wide Web – mise à disposition publique par Tim Berners-Lee.
  • 1995 : HTML 2.0 – premiers standards.
  • 2004 : Web 2.0 – Interaction, réseaux sociaux, AJAX.
  • 2010 : HTML5 – multimédia intégré, sans plugins.
  • 2020 : Web 3.0 – décentralisation, blockchain, IA.

🔗 URL & Protocoles HTTP

Structure d’une URL : https://www.exemple.com/page?param=1

  • HTTPS : Protocole sécurisé
  • Nom de domaine : www.exemple.com
  • Chemin : /page
  • Paramètre : ?param=1

Codes de statut HTTP :

  • 200 OK – Tout va bien
  • 301 Moved Permanently – Redirection permanente
  • 302 Found – Redirection temporaire
  • 404 Not Found – Page non trouvée

🌍 Noms de Domaine & DNS

Principe : Adresse lisible par l’humain reliée à une IP serveur.

Cycle de vie : Réservation → Délégation → Expiration

Acteurs : ICANN, registraires (OVH, Gandi, GoDaddy…)

DNS : Système qui traduit les noms de domaines en adresses IP.

⚙️ Standards du Web

Le W3C (World Wide Web Consortium) crée les standards comme HTML, CSS, etc.

Ces standards garantissent la compatibilité entre navigateurs et évitent le chaos technique.

🔐 Sécurité sur le Web

Menaces courantes :

  • Phishing – Tentatives de vol d’identifiants
  • Malware – Logiciels malveillants
  • Brute Force – Tentatives massives de connexion
  • Scraping – Extraction de contenu automatique

Bonnes pratiques :

  • HTTPS obligatoire
  • Mots de passe forts & double authentification
  • Mise à jour régulière des systèmes
  • Limiter les tentatives de connexion
